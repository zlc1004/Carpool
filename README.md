# CarpSchool

CarpSchool is a React Native mobile application for student ride sharing, built with Expo and React Native Paper. This app connects students traveling to/from school campus to easily coordinate safe and verified ride shares.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install CarpSchool dependencies.

```bash
npm install
```

## Usage

```bash
# Start the development server
npx expo start

# Run on iOS simulator
npx expo start --ios

# Run on Android emulator
npx expo start --android

# Generate app icons from AppIcon.icon
./generate-icons.sh

# Create development placeholder icons
./create-placeholder-icons.sh
```
