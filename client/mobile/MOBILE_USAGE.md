## ğŸš— Creating a Ride

### Driver Setup
1. Go to "My Rides" and tap "+"
2. Fill in ride details:
   - **Origin** - Starting location (use map picker)
   - **Destination** - End location
   - **Date & Time** - When you're departing
   - **Available Seats** - How many riders you can take
   - **Notes** - Any special instructions

3. Set preferences:
   - **Price** - Optional cost per rider
   - **Recurring** - If this is a regular ride
   - **Restrictions** - Gender, school, grade preferences

4. Tap "Create Ride"

### Managing Ride Requests
When someone requests to join your ride:
1. You'll get a push notification
2. Open the notification or go to "My Rides"
3. View the requester's profile
4. **Accept** or **Decline** the request
5. Accepted riders will be notified automatically

## ğŸ†˜ Troubleshooting

### Common Issues

**Can't receive notifications:**
1. Check app notification permissions in device settings
2. Ensure the app is updated to the latest version
3. Try signing out and back in
4. Contact support if issues persist

**Location not working:**
1. Enable location permissions for the app
2. Check that location services are on
3. Try restarting the app
4. Ensure you have a good internet connection

**Can't find rides:**
1. Expand your search radius
2. Try different departure times
3. Check that your origin/destination are correct
4. Make sure your filters aren't too restrictive

**Messages not sending:**
1. Check your internet connection
2. Make sure you haven't been blocked
3. Try force-closing and reopening the app
4. Contact support for persistent issues
